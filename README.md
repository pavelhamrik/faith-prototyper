# faith-prototyper
OSX prototype exporter for Faith CCG.

## Uses
[ZipZap compression/decompression utility library](https://github.com/pixelglow/ZipZap)

[SWXMLHash XML parsing library](https://github.com/drmohundro/SWXMLHash)
